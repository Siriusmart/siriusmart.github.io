Control unit

Arthmetic and logic unit (ALU)

  • Logical operations
  • Shift operations

Registers

A register is a very fast memory location within the CPU.

Register Description
Memory address register (MAR) Holds address of the instruction or data
Memory data register (MDR) Holds data or a program
Program counter (PC) Holds address of the next instruction
Accumulator (A) Holds the result of the last operation

  • Speed which processor operates in
  • Hertz (Hz) is the number of times the cycle can run in one second
