Data Representation

Unit Description
Bit (b) 1 or 0
Nibble 4 bits
Byte (B) 8 bits
Kilobyte (KB) 1024 bytes
Megabyte (MB) 1024 kilobytes
Gigabyte (GB) 1024 megabytes
Terabyte (TB) 1024 gigabytes (according to hardware manufacturers it’s 1000GB, and it’s obviously a scam)
Petabyte (PB) 1024 terabytes

Binary

  1. First digit is 2⁰ = 1
  2. 2ⁱ = 2
  3. 2² = 4
  4. …

With n bits can store any number from 0 to 2ⁿ - 1

Most significant bit

  • The most left bit, no matter it is 0 or 1
  • The most right bit is the least significant bit

Hexadecimal

  • 0123456789ABCDEF (base 16)
  • 4 nibbles = 1 hexadecimal digit
  • More readable to human

Binary shifts

  • Left shift: Multiply by 2 (might loose the most significant bit)
  • Right shift: Divide by 2 (might loose the least significant bit but it doesn’t really matter)

Characters

  • A character set maps a binary value to a character
Character set Bits per character
Ascii 7 bits
Extended ASCII 8 bits (1 byte)
Unicode (UTF-8 is the greatest invention) 16 - 32 bits
