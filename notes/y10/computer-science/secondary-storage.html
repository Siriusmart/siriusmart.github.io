Secondary Storage

  • Non-volatile
  • Hold much more data than main memory
  • Cheap

Magnetic devices

  • Magnetic disks are read with a moving head inside the disk drive
  • Slow compared to SSD because it contains moving parts
  • Vulnerable to magnetic fields
Property Value
Thickness 2.5 - 3.5 inches
Capacity in the TBs
Speed 100 - 200 MB/s
Durability Vulnerable to movement, heat, and magnetic fields
Reliability Extremely reliable

Solid State Drives (SSD)

  • More reliable - less susceptible to shock and damage (they have no moving parts)
  • Lighter in weight
  • Faster than HDDs
  • Lower power consumption
  • Does not heat up like HDDs
  • No moving parts - very thin
  • More expensive than HDDs
Property Internal SSD USB stick Memory Card
Capacity Up to 4TB 16MB - 256GB 32GB - 1TB
Speed Up to 3.5GB/s 3 - 100MB/s Depends on type
Durabiiliy Not damaged by shock or heat Same Same
Reliability Extremely reliable Same Same

Note

Limited times that data can be written before it wears off

Optical devices

  • Lasers read and write data to the optical disk
  • Have moving parts
  • Easy to be damaged
Property CD DVD Blu-ray
Capacity 640MB 4.7GB (8.5GB if dual layer) 25GB (50GB if dual layer)
Speed 7.6MB/s 16MB/s 50MB/s
Durability Affected by exposure to sunlight, more physical damage, degrades over time Same Same
