Streams

Output streams

Take a look at cout

cout << "Hello World!";

Here you are actually pushing the string “Hello World!” to the output buffer, the buffer will then be printed to the terminal.

Besides pushing output to cout, you can also push output to ofstream (files)

#include <fstream>

// snip...

ofstream myFile("myFile.txt");

if (myFile) {
    myFile << "Hello World!"; // this line
    myFile.close();
} else {
    cout << "Could not open file" << endl;
}

In the above example, we created a buffer for the file myFile.txt, then we added “Hello World!” to the buffer, and when we close the file, the changes will be written to it

Input streams

In the same sense input streams are a buffer for the input (usually a string), and reading from the buffer removes the read part from the buffer.

#include <string>

// snip...

string myString;
cin >> myString; // takes one word (until a space) from the standard input

getline(cin, myString); // takes one line from the standard input (until line break '\n')

This rule also applies to ifstream for files

#include <fstream>
#include <string>

// snip...

ifstream myFile("myFile.txt");

if (myFile) {
    
    myFile >> myString; // takes one word (until a space) from the file
    getline(myFile, myString); // takes one line from the file (until line break '\n')

} else {
    cout << "Could not open file" << endl;
}
