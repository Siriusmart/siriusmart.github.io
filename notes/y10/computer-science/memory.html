Memory

Random Access Memory (RAM)

  • Main memory
  • Volatile (when powered off data is loss)
  • Temporary storage for programs and data while the program is running
  • CPU cannot go directly to secondary storage

Virtual Memeory (VRAM)

  • Part of the secondary storage to behave like main memory
  • Is slower because swapping takes time
  1. RAM gets full
  2. Sections are swapped out to secondary storage, freeing up space for in RAM (paging)

Read Only Memory (ROM)

  • Cannot write to it once created
  • Non-volatile
  • Holds the bootstrap for the computer
RAM ROM
Volatile - data is lost when the power is turned off (temporary memory) Non-volatile - data is not loss when the power is turned off (permanent memory)
Stores data that is currently in use Stores BIOS/bootstrap loader used at startup
Can be read and write to Can only be read and not written to
