Array and vectors

  • An array has a fixed size and cannot be changed
  • An array starts at index 0 (the first item is 0)
// declare an array
// type name[size] = items;

int arr[2] = {1, 2};
cout << arr[1] << endl; // -> 2
  • A vector is a class that need to be #included
  • A vector can have a variable size
  • You cannot return a vector from a function
#include <vector>

using namespace std;

// snip...

// declare a vector

vector<int> v; // a blank vector
v.push_back(1); // add an item to the vector
int x = v.pop_back(); // remove the last item and returns it, x is now 1

v.size(); // length of vector is 0

String

  • String is an array of characters
  • Include string
#include <string>

using namespace std;

// snip...

string greet = "Hello world!";

greet[0]; // first character of the string which is 'H'
greet[greet.length() - 1]; // last character of the string which is '!'
